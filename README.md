# 🦍概要
ゴリラを駆逐し渋谷の治安を守るシューティングゲーム。ゴリラを12匹倒すと渋谷を荒らす張本人（ボス）が出現し，そいつを倒すとゲームクリアです。

# 🎮操作方法
### zボタン
ゲーム開始

### 右ボタン，左ボタン
左右移動

### スペースキー
ジャンプ。敵の接触から身を守ることができる。

### 下ボタン
![Punching](/src/img/punching.png)
敵を攻撃。
